# API 说明

本文记录当前 API 分组、鉴权规则、关键请求和 schema contract。字段细节以 controller、dto、service 和 `src/shared/api/*` 为准。

## 基础信息

```text
本地后端：http://localhost:3000/api
前端配置：VITE_API_BASE_URL=http://localhost:3000/api
Token key：lowcode_editor_token
鉴权头：Authorization: Bearer <accessToken>
```

统一错误响应包含：

```json
{
  "statusCode": 400,
  "code": "PAGE_SCHEMA_INVALID",
  "message": "错误信息",
  "path": "/api/pages/1",
  "method": "PATCH",
  "requestId": "request-id",
  "timestamp": "2026-04-28T00:00:00.000Z"
}
```

## 鉴权边界

公开接口：

- `POST /api/auth/register`
- `POST /api/auth/login`
- `GET /api/public/pages/:publicId`

受保护接口：

- 项目、页面、版本、发布、素材、模板等业务接口需要 JWT。
- 管理员接口需要 JWT + `AdminGuard`。

项目角色：

| 角色 | 能力 |
| --- | --- |
| owner | 管理项目、成员、审计、页面、版本、发布 |
| editor | 读项目，创建/保存/发布/回滚页面 |
| viewer | 只读项目、页面、版本 |

## Auth API

| 方法 | 路径 | 说明 |
| --- | --- | --- |
| POST | `/api/auth/register` | 注册并返回 token |
| POST | `/api/auth/login` | 登录并返回 token |
| GET | `/api/auth/me` | 获取当前用户 |

注册：

```json
{
  "email": "user@example.com",
  "username": "username",
  "password": "password123"
}
```

登录：

```json
{
  "account": "username or email",
  "password": "password123"
}
```

响应核心结构：

```json
{
  "accessToken": "jwt-token",
  "user": {
    "id": 1,
    "email": "user@example.com",
    "username": "username",
    "role": "user",
    "status": "active"
  }
}
```

## Projects API

| 方法 | 路径 | 权限 | 说明 |
| --- | --- | --- | --- |
| GET | `/api/projects` | 项目成员 | 当前用户可访问项目 |
| POST | `/api/projects` | 登录用户 | 创建项目，自动成为 owner |
| GET | `/api/projects/:id` | 项目成员 | 项目详情 |
| PATCH | `/api/projects/:id` | owner | 更新项目 |
| DELETE | `/api/projects/:id` | owner | 删除项目 |
| GET | `/api/projects/:id/members` | 项目成员 | 成员列表 |
| POST | `/api/projects/:id/members` | owner | 添加成员 |
| PATCH | `/api/projects/:id/members/:memberId` | owner | 修改成员角色 |
| DELETE | `/api/projects/:id/members/:memberId` | owner | 移除成员 |
| GET | `/api/projects/:id/audit-logs` | owner | 项目审计 |

成员角色只支持新增或调整为 `editor/viewer`，不能通过成员接口新增或移除 owner。

## Pages API

| 方法 | 路径 | 权限 | 说明 |
| --- | --- | --- | --- |
| GET | `/api/projects/:projectId/pages` | 项目成员 | 页面列表 |
| POST | `/api/projects/:projectId/pages` | owner/editor | 创建页面 |
| GET | `/api/pages/:id` | 项目成员 | 页面详情 |
| PATCH | `/api/pages/:id` | owner/editor | 更新页面和 schema |
| DELETE | `/api/pages/:id` | owner/editor | 删除页面 |
| GET | `/api/pages/:id/versions` | 项目成员 | 版本列表 |
| POST | `/api/pages/:id/rollback` | owner/editor | 回滚到版本 |
| DELETE | `/api/pages/:id/versions/:versionId` | owner/editor | 删除版本记录 |
| POST | `/api/pages/:id/publish` | owner/editor | 发布页面 |
| POST | `/api/pages/:id/unpublish` | owner/editor | 取消发布 |
| GET | `/api/public/pages/:publicId` | 公开 | 读取发布快照 |

创建页面：

```json
{
  "name": "首页",
  "routePath": "/home"
}
```

保存页面：

```json
{
  "name": "首页",
  "routePath": "/home",
  "schema": {
    "schemaVersion": "1.0.0",
    "pageId": 1,
    "components": [
      {
        "id": 1,
        "name": "Page",
        "props": {},
        "desc": "页面",
        "children": []
      }
    ],
    "metadata": {
      "updatedAt": "2026-04-28T00:00:00.000Z"
    }
  }
}
```

回滚：

```json
{
  "versionId": 1
}
```

## Admin API

统一前缀：`/api/admin`。

| 方法 | 路径 | 说明 |
| --- | --- | --- |
| GET | `/api/admin/overview` | 平台概览 |
| GET | `/api/admin/users` | 用户列表 |
| PATCH | `/api/admin/users/:id/status` | 启用/禁用用户 |
| GET | `/api/admin/projects` | 项目列表 |
| PATCH | `/api/admin/projects/:id/status` | 启用/禁用项目 |
| GET | `/api/admin/published-pages` | 已发布页面 |
| POST | `/api/admin/pages/:id/unpublish` | 管理员取消发布 |
| GET | `/api/admin/audit-logs` | 全局审计 |

管理员操作会写入审计日志。禁用项目会阻断普通用户访问，并取消该项目下公开发布页。

## Schema Contract

组件节点：

```ts
{
  id: number;
  name: string;
  props: any;
  styles?: CSSProperties;
  desc: string;
  children?: Component[];
  parentId?: number;
}
```

页面 schema：

```ts
{
  schemaVersion: CURRENT_SCHEMA_VERSION;
  pageId?: number;
  components: Component[];
  metadata?: Record<string, unknown>;
}
```

关键规则：

- 前端保存前校验 `components`。
- 后端创建、保存、发布、回滚前都会先迁移再校验。
- `Page.schema` 是当前草稿。
- `PageVersion.schema` 是保存、回滚或发布时的快照。
- 公开页只读取 `publishedVersionId` 指向的版本快照。
