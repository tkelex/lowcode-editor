# 事件能力矩阵

本文档记录当前注册表中已经落地的事件与动作能力。后续新增物料时，需要同步更新 `src/editor/registry/component-config.tsx` 和本文档。

## 动作类型

| actionType | 名称 | 说明 |
| --- | --- | --- |
| `toast` | 消息提示 | 展示成功、失败、警告或普通提示。 |
| `url` | 页面跳转 | 当前窗口或新窗口打开站内/外部链接。 |
| `componentAction` | 组件方法 | 调用目标组件暴露的方法，例如弹窗打开/关闭、表单提交/重置。 |
| `componentControl` | 组件联动 | 显示/隐藏、启用/禁用、设置/清空值、打开/关闭弹窗、提交/重置表单。 |
| `confirm` | 确认弹窗 | 先确认，再执行确认/取消分支动作列表。 |
| `condition` | 条件执行 | 根据表达式或常用条件模板执行真假分支动作列表。 |
| `http` | HTTP 请求 | 支持鉴权、headers/body 模板、响应/错误字段映射。 |
| `setComponentProps` | 设置属性 | 用 JSON 修改目标组件 props。 |
| `setComponentStyles` | 设置样式 | 用 JSON 修改目标组件 styles。 |
| `setVariable` | 设置变量 | 把固定 JSON 值或表达式结果写入页面运行态变量。 |
| `custom` | 自定义 JS | 编辑器预览可用，公开发布页禁用。 |

## 事件类别默认动作

| 事件类别 | 默认动作 |
| --- | --- |
| `ui` | `toast`、`url`、`componentAction`、`componentControl`、`confirm`、`condition`、`http`、`setComponentProps`、`setComponentStyles`、`setVariable`、`custom` |
| `value` | `toast`、`componentAction`、`componentControl`、`condition`、`http`、`setComponentProps`、`setComponentStyles`、`setVariable`、`custom` |
| `submit` | `toast`、`url`、`componentAction`、`componentControl`、`confirm`、`condition`、`http`、`setComponentProps`、`setComponentStyles`、`setVariable`、`custom` |
| `overlay` | `toast`、`componentAction`、`componentControl`、`confirm`、`condition`、`http`、`setComponentProps`、`setComponentStyles`、`setVariable`、`custom` |
| `lifecycle` | `toast`、`componentAction`、`componentControl`、`condition`、`http`、`setComponentProps`、`setComponentStyles`、`setVariable`、`custom` |

## 组件方法

| 组件 | 方法 | 说明 |
| --- | --- | --- |
| Modal | `open`、`close` | 打开或关闭弹窗。 |
| Drawer | `open`、`close` | 打开或关闭抽屉。 |
| Form | `submit`、`reset`、`getValues`、`setValues` | 提交、重置、读取或设置表单数据。 |

## 配置规则

- 右侧事件面板只展示当前组件注册表声明的事件。
- 添加动作时按事件 `allowedActions` 过滤动作类型。
- 事件数据提示来自 `eventDataSchema`，自定义 JS、条件表达式和 HTTP 模板都可以引用。
- 动作按列表顺序执行，可排序、复制、删除、启用或禁用。
- `confirm` 和 `condition` 的嵌套分支也使用完整动作列表。
- `setVariable` 可引用 `event`、`context`、`variables`、`args`，组件属性可用 `{{variables.xxx}}` 绑定页面变量。
- 公开发布页必须传 `allowCustomJS={false}`，`custom` 动作不会执行。
