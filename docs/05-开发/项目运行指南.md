# 项目运行指南

本文档只保留启动项目需要的关键命令。默认使用 Windows PowerShell，并在项目根目录执行：

```powershell
cd E:\前端\nk_React\project\lowcode-editor
```

## 首次运行

### 1. 安装依赖

```powershell
npm.cmd install
npm.cmd install --prefix server
```

### 2. 准备环境变量

```powershell
Copy-Item .env.example .env
Copy-Item server/.env.example server/.env
```

### 3. 启动数据库

```powershell
docker compose up -d postgres
docker compose ps
```

### 4. 初始化数据库

```powershell
npm.cmd run prisma:generate --prefix server
npm.cmd run prisma:migrate --prefix server
```

## 日常启动

### 1. 启动数据库

```powershell
docker compose up -d postgres
```

### 2. 启动后端

新开一个终端：

```powershell
npm.cmd run dev --prefix server
```

### 3. 启动前端

再新开一个终端：

```powershell
npm.cmd run dev
```

## 访问地址

```text
前端：http://localhost:5173
后端：http://localhost:3000/api
数据库：localhost:5432
```

## 常用检查

```powershell
npm.cmd run build
npm.cmd run build --prefix server
npm.cmd run check
npm.cmd run test:e2e
npm.cmd run smoke:api
```

`test:e2e` 会自动启动前端开发服务器，并使用 mock API 验证前端主链路。首次运行如缺少浏览器，可执行：

```powershell
npx.cmd playwright install chromium
```

`smoke:api` 需要先启动数据库和后端，用来验证真实后端接口链路。

## 停止项目

前端和后端终端按：

```text
Ctrl + C
```

停止数据库：

```powershell
docker compose down
```
